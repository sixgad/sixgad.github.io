2.先了解下webpack的基本结构

Webpack 是一个前端资源加载/打包工具。它将根据模块的依赖关系进行静态分析，然后将这些模块按照指定的规则生成对应的静态资源。

整体上看是一个自执行函数，传入的参数一般为object或者数组（内容都是function定义的一个一个方法）

必不可少的加载器，关键字exports，一般长这样

    function o(t) {
        if (n[t])
            return n[t].exports;
        var i = n[t] = {
            i: t,
            l: !1,
            exports: {}
        };
        return e[t].call(i.exports, i, i.exports, o),
        i.l = !0,
        i.exports
    }
